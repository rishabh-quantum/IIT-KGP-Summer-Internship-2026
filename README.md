## Project Description
This project focused on the theoretical foundations of quantum error correction using surface codes. The work involved studying stabilizer codes, syndrome measurements, logical qubits, and the principles behind fault-tolerant quantum computation.
The project also includes implementing selected concepts using Qiskit to strengthen the connection between theoretical understanding and computational simulations.
